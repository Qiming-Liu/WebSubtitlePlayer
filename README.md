## What is this
> A Local Web .srt (Subtitle) File Player. You can pause, play and adjust the font size.

## Package
* Vue 3
* Vite

## Development
Development and test in live load
```shell
    npm install
    npm run dev
```
Build
```shell
    npm install
    npm run build
```
